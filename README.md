# RTRP-PROJECT
Machine learning-based system analyzing women’s safety in Indian cities using X (Twitter) data. Applies NLP and sentiment analysis (RF, SVM, TextBlob) to classify tweets and provide city-wise insights with visualizations, helping identify safety concerns and support data-driven decisions.

# Women Safety Analysis using Machine Learning

![Project Banner](https://via.placeholder.com/1200x300?text=Women+Safety+Analysis+ML)

## Overview

This project analyzes women’s safety in Indian cities using machine learning and natural language processing on X (Twitter) data. It classifies tweets into positive, negative, and neutral sentiments to identify trends and safety concerns across different cities.

---

## Features

- Automated tweet preprocessing (cleaning and tokenization)
- Sentiment analysis using Machine Learning and TextBlob
- City-wise safety analysis
- Graphical visualization (pie charts and bar graphs)
- User-friendly interface (Tkinter/Flask)
- Scalable for large datasets
