<p align="center">

<h1>Women Safety Analysis using Machine Learning</h1>

<h2>Overview</h2>
<p>
This project analyzes women’s safety in Indian cities using machine learning and natural language processing on X (Twitter) data. It classifies tweets into positive, negative, and neutral sentiments to identify trends and safety concerns across different cities.
</p>

<hr>

<h2>Features</h2>
<ul>
  <li>Automated tweet preprocessing (cleaning and tokenization)</li>
  <li>Sentiment analysis using Machine Learning and TextBlob</li>
  <li>City-wise safety analysis</li>
  <li>Graphical visualization (pie charts and bar graphs)</li>
  <li>User-friendly interface (Tkinter/Flask)</li>
  <li>Scalable for large datasets</li>
</ul>
